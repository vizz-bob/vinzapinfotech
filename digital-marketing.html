📋 SEO On-Page Checklist 

SEO ON-PAGE OPTIMIZATION CHECKLIST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ Title Tag        — 50–60 chars, keyword first
✅ Meta Description — 150–160 chars, includes CTA
✅ H1 Tag           — Only 1 per page, has keyword
✅ H2/H3 Tags       — Subheadings, include LSI keywords
✅ URL Slug         — Short, keyword-rich, no dates
✅ Image Alt Text   — Descriptive, includes keyword
✅ Internal Links   — 3–5 links to related pages
✅ External Links   — 1–2 links to authoritative sites
✅ Keyword Density  — 1–2% (don't stuff!)
✅ Content Length   — Min 1000 words for blogs
✅ Schema Markup    — Article, FAQ, HowTo schema
✅ Page Speed       — <3 seconds load time
✅ Mobile Friendly  — Test on Google Mobile Test

📋 Google Ads Quick Reference 

GOOGLE ADS KEYWORD MATCH TYPES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Broad Match     →  digital marketing course
                   (shows for related searches)
Phrase Match    → "digital marketing course"
                   (phrase must appear in query)
Exact Match     → [digital marketing course]
                   (exact query only)
Negative Match  → -free
                   (exclude this keyword)

AD COPY FORMULA (RSA)
─────────────────────
Headline 1: Include Main Keyword (30 chars)
Headline 2: Unique Value Proposition (30 chars)
Headline 3: Call to Action (30 chars)
Description: Benefit + Feature + CTA (90 chars each)

QUALITY SCORE FACTORS
──────────────────────
• Expected CTR          (40%)
• Ad Relevance          (40%)
• Landing Page Exp.     (20%)

Target: Quality Score ≥ 7/10
Higher QS = Lower CPC + Better Positions

BIDDING STRATEGIES
───────────────────
Manual CPC      → Full control, good for new campaigns
Target CPA      → Set cost per conversion goal
Target ROAS     → Set return on ad spend goal
Maximize Clicks → Best for traffic campaigns

📋 Meta Ads Campaign Structure 

META ADS CAMPAIGN STRUCTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━
Campaign Level → Set Objective (Awareness/Traffic/Leads/Sales)
  Ad Set Level → Audience + Budget + Schedule + Placement
    Ad Level   → Creative (Image/Video/Carousel) + Copy

AUDIENCE TARGETING OPTIONS
───────────────────────────
Core Audience    → Age, Location, Interests, Behaviors
Custom Audience  → Website visitors, Email list, App users
Lookalike        → Find people similar to your customers
                   (1%=most similar, 10%=broader)

META ADS CAMPAIGN CHECKLIST
─────────────────────────────
✅ Install Meta Pixel on website
✅ Verify domain in Business Manager
✅ Create Custom Conversion events
✅ Test with ₹200-500/day budget first
✅ Run A/B test on creatives (2-3 ads per set)
✅ Wait 7 days before optimizing (learning phase)
✅ Check frequency — keep below 3 for cold audience
✅ Refresh creatives every 2-3 weeks (ad fatigue)

AD COPY FORMULA
───────────────
Hook (1 line)  → Grab attention immediately
Problem        → Identify the pain point
Solution       → Your product/service
Social Proof   → "10,000+ students enrolled"
CTA            → "Click to Enroll Free"

📋 Content Marketing Templates 

BLOG POST STRUCTURE
━━━━━━━━━━━━━━━━━━
1. Headline      → Include keyword, number, power word
                   Example: "7 Free Digital Marketing Tools in 2025"
2. Introduction  → Hook + Problem + Promise (100-150 words)
3. Table of Contents (for posts >2000 words)
4. Body Sections → H2 headings, bullet points, examples
5. Images/Visuals → Every 300-400 words
6. Conclusion    → Summary + CTA
7. FAQ Section   → Targets featured snippets

HEADLINE FORMULAS THAT WORK
────────────────────────────
• How to [Achieve Result] in [Timeframe]
• [Number] Ways to [Solve Problem]
• The Ultimate Guide to [Topic] in [Year]
• [Number] [Topic] Mistakes You're Making
• Why [Common Belief] is Wrong (And What to Do)

CONTENT TYPES BY FUNNEL STAGE
───────────────────────────────
TOFU (Awareness)  → Blog posts, infographics, reels
MOFU (Interest)   → Case studies, webinars, comparisons
BOFU (Decision)   → Testimonials, demos, free trials

📋 YouTube Channel Optimization Checklist 

YOUTUBE VIDEO OPTIMIZATION
━━━━━━━━━━━━━━━━━━━━━━━━━
✅ Title: 60 chars max, keyword in first 3 words
✅ Description: 250+ words, keyword in first 2 lines
                Add timestamps, links, social handles
✅ Tags: 10-15 tags, mix broad & specific
✅ Thumbnail: 1280×720px, high contrast, face + text
✅ Cards: Add at 70% mark pointing to related video
✅ End Screen: Last 20 sec — subscribe + video CTA
✅ Chapters: Add timestamps in description (00:00 Intro)
✅ Pinned Comment: First comment = keywords + links
✅ Subtitles: Upload SRT file for accessibility + SEO

YOUTUBE SHORTS FORMULA
─────────────────────
0-3 sec:  Hook (question, bold statement, or surprise)
3-15 sec: Context / setup
15-50 sec: Value / payoff
50-60 sec: CTA ("Follow for more DevOps tips!")

BEST POSTING SCHEDULE
──────────────────────
Long videos: 2–3 per week (Tue, Thu, Sat)
Shorts:      Daily or 5x/week
Consistency > Frequency

EMAIL WELCOME SEQUENCE (7 DAYS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Day 0:  Welcome + deliver lead magnet
Day 1:  Your story + why you help
Day 2:  Best free resource (tutorial/blog)
Day 3:  Social proof (testimonial/case study)
Day 4:  Pain point email (identify problem)
Day 5:  Solution email (introduce your offer)
Day 6:  FAQ / objection handling
Day 7:  Soft pitch (offer + deadline)

EMAIL SUBJECT LINE FORMULAS
─────────────────────────────
Curiosity:     "You won't believe this SEO mistake..."
Urgency:       "Last 24 hours: Free course ending tonight"
Personalized:  "Hey [Name], your action plan is ready"
Numbers:       "5 tools every digital marketer needs"
Question:      "Are you making this Instagram mistake?"

KEY EMAIL METRICS TO TRACK
────────────────────────────
Open Rate      → 20-25% is good (30%+ excellent)
Click Rate     → 2-3% is average (5%+ excellent)
Unsubscribe    → Keep below 0.5%
Bounce Rate    → Keep below 2%
